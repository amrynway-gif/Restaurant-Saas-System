# لوحة التحكم — ألوان مريحة على غرار نظام المنار

## التقنية المستخدمة في موقع المنار (almnarhome.com)

من خلال الرابط والصور المرفقة:

- **الإطار:** **Next.js** (يظهر من مسارات الصور `/_next/image` في الموقع).
- **الواجهة:** واجهة React مع تحميل ديناميكي («جاري التحميل…»).
- **التصميم:** لوحة تحكم داكنة بألوان هادئة (أزرق غامق/رمادي أزرق)، بنفسجي للعناصر الرئيسية، وأخضر للعنصر النشط في القائمة.

مشروعنا يستخدم نفس التقنية (Next.js + React + Tailwind + مكوّنات حديثة)، لذا يمكن الوصول لمستوى سلاسة ووضوح مشابه عبر التصميم والألوان.

---

## ما تم تطبيقه في لوحة تحكمنا

### 1. ثيم داكن بألوان مريحة (على غرار المنار)

في **الوضع الداكن** (`data-theme="dark"`) تم ضبط متغيرات التصميم في `globals.css` لتعطي إحساساً قريباً من لوحات التحكم الحديثة:

| الاستخدام | اللون / المتغير |
|-----------|------------------|
| خلفية أساسية | أزرق غامق Slate `#0f172a` |
| خلفية الكروت/السطح | `#1e293b` |
| سطح ثانوي | `#334155` |
| البراند (أزرار رئيسية، عناوين) | بنفسجي `#8b5cf6` |
| **العنصر النشط في القائمة الجانبية** | أخضر `#10b981` مع خلفية `#064e3b` |
| النص الرئيسي | أبيض ناعم `#f8fafc` |
| النص الثانوي والـ Muted | رمادي فاتح `#cbd5e1` و `#94a3b8` |
| الحدود | رمادي أزرق خفيف `#334155` |

### 2. الشريط الجانبي (Sidebar)

- الرابط **النشط** (الصفحة الحالية) يستخدم الآن:
  - `--sidebar-active` (أخضر)
  - `--sidebar-active-bg` (خلفية أخضر غامق)
- باقي الروابط: لون ثانوي مع hover بلون السطح الثانوي.

### 3. النتيجة

- خلفيات داكنة مريحة للعين (أزرق غامق بدل البنفسجي الكثيف).
- تمييز واضح للصفحة الحالية بلون أخضر.
- بنفسجي للبراند والأزرار المهمة.
- حدود وظلال ناعمة لشكل حديث وواضح.

لرؤية التغييرات: تأكد أن الوضع الداكن مفعّل في التطبيق (مبدل الثيم في الهيدر أو `data-theme="dark"` في الـ layout).
